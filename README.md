# TrueworkCLI
Hackday project to make a CLI for Truework

Supports:
- Verifications API
  - List
  - Get
  - Create
- Company Search

`$ node index.js verifications create --purpose employment -f Felix -l Sargent --ssn 111-111-1111 -c Truework --type employment`
